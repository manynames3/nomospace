# nomospace Privacy

nomospace is designed as a local-first Mac utility.

## Local data

nomospace scans local folder paths and file sizes to produce storage findings. Cleanup history is stored locally on the same Mac using app preferences.

Users can clear cleanup history from the History screen. Exported audit reports are saved only where the user chooses.

## No upload behavior

The current app does not upload file names, file contents, paths, browser data, cleanup history, or scan results to a server.

## Full Disk Access

macOS may block access to Desktop, Documents, Downloads, app containers, and parts of Library. Full Disk Access makes the scan more complete. It does not give nomospace permission to send data anywhere.

## Cleanup behavior

nomospace moves selected items to macOS Trash. It does not empty Trash automatically.

## Browser data

Browser-related findings are marked `Review`. Removing browser profile data can affect login state, extensions, and local web app state. nomospace does not inspect cookies, saved passwords, or browser secrets.
