# Roadmap

This product should stay narrow: explain hidden Mac storage and make cleanup safer.

## Now

- Local-only storage audit.
- Risk-labeled findings.
- Trash-first cleanup.
- Local cleanup history.
- Sharable Link and Save PDF report actions.
- Evaluation mode with local access-code unlock.
- Static sales page.

## Before first paid users

- Developer ID signed and notarized release.
- DMG or signed ZIP distribution.
- Test on 5-10 real Macs with different user profiles.
- Tune storage rules from real audit results.
- Replace the local beta code with real per-customer payment/licensing.

## Later

- Expanded rule library.
- In-app update mechanism.
- Better before/after comparison.
- Optional report destinations beyond local PDF.

## Avoid

- Antivirus positioning.
- RAM or performance booster claims.
- Broad app uninstaller workflows.
- Auto-deleting personal files.
- Cloud sync or accounts before there is a clear reason.
