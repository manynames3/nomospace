# Sellable MVP Scope

This repo is aiming for a narrow paid utility, not a generic Mac cleaner.

## MVP promise

Find hidden Mac storage that normal cleaners miss, explain what created it, and move selected safe items to Trash.

## Keep

- Local-only storage audit.
- Risk-labeled findings.
- Trash-first cleanup.
- First-run trust and Full Disk Access guidance.
- Cleanup receipt/history.
- Sharable Link and local PDF audit report.
- A bundled, testable rule library.

## Cut for now

- Enterprise device management.
- Antivirus/security scanning.
- Duplicate-file detection.
- RAM or performance boosting.
- Broad app uninstall workflows.
- Cloud accounts or collaboration.

## Before charging money

- Notarize the app.
- Ship a Developer ID signed release artifact using the bundled app icon.
- Test on at least 5 real Macs with different user profiles.
- Add paid unlock through StoreKit or a simple license provider.
- Validate exported reports with helpers/consultants who need before/after proof.
